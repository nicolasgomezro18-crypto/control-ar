Excel y Reportes

Sincronización bidireccional con archivo Excel

📂 Importar / Exportar

Carga el archivo Excel para sincronizar los datos. La aplicación leerá las hojas y poblará todos los módulos. Al exportar, se generará un archivo con todos los datos actualizados.

📊 Estructura del Excel

  • 📄 CICLOS REALIZADOS AL DIA — Fecha · Ciclo realizado · Color
  • 📄 CICLOS PENDIENTES — Pendientes para día siguiente
  • 📄 PROGRAMACION TURNOS — Turnos generados
  • 📄 INVENTARIO INSUMOS — Movimientos de insumos
  • 📄 CONSUMO POR CICLO — Consumo asociado a ciclos
  • 📄 RESUMEN — Indicadores generales

💾 Backup local

Los datos se guardan en memoria del navegador mientras la pestaña está abierta. Para persistencia entre sesiones, descarga el JSON o el Excel.

⚙️ Configuración

Pega tus credenciales de Google Cloud y el ID de tu hoja de cálculo. Solo necesitas hacer esto una vez — la app las recordará.

📋 Cómo obtener las credenciales

Sigue la guía detallada incluida en el archivo GUIA_GOOGLE_SHEETS_SETUP.md. Resumen:

  1. Crea un proyecto en Google Cloud Console
  2. Habilita Google Sheets API
  3. Configura la pantalla de consentimiento OAuth (modo "Externo")
  4. Crea un OAuth Client ID tipo "Aplicación web"
  5. Crea una API Key restringida a Google Sheets API
  6. Agrega tu URL (la de GitHub Pages) a los orígenes autorizados
  7. Agrega tu correo como "usuario de prueba"
  8. Pega los 3 valores arriba y conecta

🔄 Cómo funciona la sincronización

  • ✅ Auto-sync (activado por defecto): cada cambio en la app se sube a Sheets en ~2 segundos
  • ✅ Subir todo: reemplaza el contenido de Sheets con los datos locales
  • ✅ Bajar todo: reemplaza los datos locales con los de Sheets
  • ⚠️ Conflictos: la última escritura gana — coordina con tu equipo si trabajan al tiempo
  • 📡 Sin internet: la app sigue funcionando, sincroniza cuando vuelvas a tener conexión
